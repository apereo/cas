# Storage - CAS Metrics

CAS metrics may be routed to varying types of databases for 
storage and analytics. The following options are available:

- Simple (In Memory)
- Graphite
- Ganglia
- JMX
- Atlas
- Signal FX
- Statsd
- InfluxDb
- Prometheus
- Wavefront
- New Relic
- CloudWatch
- ...
